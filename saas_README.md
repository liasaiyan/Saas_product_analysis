# SaaS Product Analytics

An end-to-end product analytics project for a simulated SaaS subscription application, built across two tools to reflect a realistic analyst workflow.

---

## Project Overview

| Detail | Info |
|---|---|
| **Product** | Simulated SaaS subscription app |
| **Period** | January – June 2024 |
| **Dataset** | 1,200 users, subscriptions, and user events |
| **Tools** | Python (Pandas, Matplotlib, Seaborn) · SQL (SQLite / PostgreSQL) |

---

## Key Questions Answered

1. Where are users coming from and which channel converts best?
2. How is Monthly Recurring Revenue (MRR) trending?
3. Which plan has the highest churn rate and why?
4. What features do users engage with most?
5. How does retention vary across signup cohorts?
6. What does the free-to-paid conversion funnel look like?

---

## Key Findings

- Referral channel produces the highest free-to-paid conversion rate
- Starter plan has the highest churn — users likely underusing the product before cancelling
- Enterprise plan retains the best — high contract value correlates with deeper onboarding
- Feature A and Feature B account for over 40% of all non-login events
- Month 1 is the steepest retention drop-off point across all cohorts
- MRR grows consistently month over month driven by Pro and Enterprise plan growth

## How to Run

### Python Notebook

```bash
pip install pandas numpy matplotlib seaborn jupyter
mkdir plots
jupyter notebook saas_analysis.ipynb
```

Run all cells. The three CSV files are generated automatically on the first run.

### SQL

Load the generated CSVs into your database, then run the queries:

```bash
sqlite3 saas.db
.mode csv
.import saas_users.csv saas_users
.import saas_subscriptions.csv saas_subscriptions
.import saas_events.csv saas_events
.read saas_analysis.sql
```

Or paste queries into [sqliteonline.com](https://sqliteonline.com) after uploading the CSVs.
